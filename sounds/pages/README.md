# Голос-гід сторінок — як записати

Кинь файл у цю папку з ім'ям `<slug>.m4a` або `.mp3` (диктофон iPhone дає m4a — ок).
Кнопка «голос» у шапці сторінки оживе сама — без правок коду.

| slug | сторінка |
|---|---|
| `137` | /137 — маніфест |
| `pro-nas` | /pro-nas — Дао і Ліза |
| `iventy` | /iventy — живі події |
| `poslugy` | /poslugy — робота з нами |
| `kazka` | /kazka — казка (гід сторінки, НЕ глави: глави живуть у `kazka/audio/`) |

Правила:
- Довжина вільна — це голосовий опис сторінки, не пед.
- Нема файла → кнопка сіра, підпис «Голос-гід: записується».
- Заміна файла = перезаписати з тим самим ім'ям, деплой підхопить.
