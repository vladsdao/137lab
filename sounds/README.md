# Звуки вулика — як завантажити свої

Кинь файл у цю папку з ім'ям `<id>.m4a` або `.mp3` (диктофон iPhone дає m4a — ок):

| id | педа |
|---|---|
| `core` | ядро 137/ЛАБ |
| `tvorchist` | ТВОРЧІСТЬ |
| `poslugy` | ВЗАЄМОДІЯ |
| `biznes` | 5Д БІЗНЕС |
| `pronas` | ПРО НАС |
| `simia` | СІМʼЯ |
| `podii` | ПОДІЇ |
| `qq` | питання-цитата |
| `qa` `qb` `qc` `qd` | відповіді A B C D |

Правила:
- **≤ 3 секунди** — довше двигун сам обріже (з м'яким фейдом).
- Нема файла → педа грає синт-ноту пентатоніки (фолбек).
- Заміна файла = просто перезаписати з тим самим ім'ям, деплой підхопить.
- Майбутнє (за потреби): `sounds.json` — адмін-мапа «педа → файл» без перейменувань.

## Голос-гід сторінок

Окрема конвенція: `sounds/pages/<slug>.m4a|mp3` — голосовий опис сторінки
(слоти у шапці внутрішніх сторінок). Див. `sounds/pages/README.md`.

## Барабан «ВІДГУКИ / РОЗДУМИ» (внизу вулика)

Стрічка живиться з `sounds/feed/feed.json`:

```json
[
  { "id": "rozdum-vlad-01", "type": "rozdum", "name": "Влад", "title": "Про тишу" },
  { "id": "olena",          "type": "vidguk", "name": "Олена" }
]
```

- `type`: `vidguk` (відгуки клієнтів) або `rozdum` (роздуми/практики Влада і Лізи).
- Порядок масиву = порядок у стрічці (нове — зверху, дописуй рядок на початок).
- Аудіо: `sounds/feed/<id>.m4a|mp3`. Нема файла → рядок «записується».
- Відгуки прилітають у Telegram (бот, функція `vidguk`) → модерація →
  поклади файл + рядок у feed.json. Правок коду не треба.
