# Звуки вулика — як завантажити свої

Кинь файл у цю папку з ім'ям `<id>.m4a` або `.mp3` (диктофон iPhone дає m4a — ок):

| id | педа |
|---|---|
| `core` | ядро 137/ЛАБ |
| `tvorchist` | ТВОРЧІСТЬ |
| `poslugy` | ВЗАЄМОДІЯ |
| `biznes` | 5Д БІЗНЕС |
| `pronas` | ПРО НАС |
| `simia` | СІМʼЯ |
| `podii` | ПОДІЇ |
| `qq` | питання-цитата |
| `qa` `qb` `qc` `qd` | відповіді A B C D |

Правила:
- **≤ 3 секунди** — довше двигун сам обріже (з м'яким фейдом).
- Нема файла → педа грає синт-ноту пентатоніки (фолбек).
- Заміна файла = просто перезаписати з тим самим ім'ям, деплой підхопить.
- Майбутнє (за потреби): `sounds.json` — адмін-мапа «педа → файл» без перейменувань.

## Голос-гід сторінок

Окрема конвенція: `sounds/pages/<slug>.m4a|mp3` — голосовий опис сторінки
(слоти у шапці внутрішніх сторінок). Див. `sounds/pages/README.md`.
