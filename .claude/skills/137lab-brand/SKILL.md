# 137lab — бренд: токены + голос

## Overview

137lab = «Лабораторія Ч/Б»: чистый лист, чернила, Fixel, физика света и материи (1/137). Точность токенов — не вкусовщина: baseline-тест показал дрейф (вес 700 вместо 600, #FAFAFA вместо #fff). Числа ниже — канон.

## Design tokens (канон, выбран Владом 2026-06-11)

```css
--paper: #ffffff;   /* фон — ЧИСТЫЙ белый, не #FAFAFA/#f5f5f5 */
--ink:   #0a0a0a;   /* текст и акцент — один цвет, Ч/Б без серых акцентов */
--font-display: 'Fixel Display';  /* вес 600 (НЕ 700), титулы */
--font-text:    'Fixel Text';     /* вес 400, body */
/* титул: clamp(46px, 12vw, 92px); letter-spacing -0.010em (НЕ -0.02); line-height 1.04 */
/* body: 17–18px; line-height 1.65 */
```

- Канвас **9:16** (база 390×844), mobile-first портрет; десктоп = тот же канвас по центру листа.
- Шрифты self-hosted: `v4/fonts/*.woff2` (Fixel, SIL OFL). Не Google Fonts.
- Инверсия для тёмных медиа: текст #ffffff на #0a0a0a, скрим-слой для читаемости.
- Лого: «137lab» латиницей — «137» Display 600 (ls −0.010em) + «lab» Display 400 вплотную, ~15px.

## Цвет: «колір як пробудження» (решение Влада 2026-06-11, вечер)

Ч/Б — скелет мира (фон, решётка, текст, пульс — всегда чернила). **Цвет рождается
ТОЛЬКО как отклик на действие ребёнка** — волны от пэдов, сигилы, заработанные грани.
Нота → цвет (ньютоновская синестезия): C красный · D оранж · E жёлтый · G синий ·
A фиолет; октава выше — светлее, ниже — глубже (`NOTE_COLOR` в hive/index.html — канон).
Красное «ВІДЧУВАЙ» (13.7%) — часть той же доктрины. NEVER: цветные фоны, цветной UI
в покое, радужный декор. Цвет = след прикосновения, не краска.

## Голос (полный гайд: `voice.md` в корне этого репо)

- **Тест стола:** показываем находки, не тянем за рукав. Глаголы: досліджуємо, показуємо, запрошуємо. NEVER: запишіться, отримайте, встигніть, пропонуємо.
- **Два голоса:** Дао (структура, ирония, инженерные метафоры) и Лиза (точность переживания). «Мы» — заработанное слово.
- **Ирония:** одна странность на страницу. Если страница смешная — проиграли; если нигде не улыбается — тоже.
- UA первичный; «Угода про вигадку» — общий ритуал с казкой Insaint (один механизм, два продукта).
- Наследуются литмусы: тест Бодрийяра, тест сцены, NEVER-лексика (вибрації, пробудження, вище я, гуру).

## Common mistakes

- Вес 700 «для солидности» → канон 600.
- «Почти белый» фон → только #ffffff.
- Трекинг на глаз → -0.010em для титулов.
- Продажные глаголы в CTA → тест стола.
