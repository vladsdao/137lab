## Правила деплоя (ЖЕЛЕЗНЫЕ)

1. **Push в main = продакшн** (Netlify auto-deploy на 137lab.xyz). Push ТОЛЬКО с явного approve Влада.
2. **Всё additive.** Никаких удалений файлов, массовых перезаписей, `git push --force` — без per-operation подтверждения Влада.
3. **`/sonora` и `/retreat` неприкосновенны** — боевые/архивные страницы, менять только по прямой просьбе Влада.
4. **Новые страницы — `noindex, nofollow`** до отдельного approve.
5. Корневой `index.html` — только с approve.
